# Contract: SEO Manifest (`frontend/seo-manifest.json`)

Generated by `tools/seo/build-articles.mjs`. Git-ignored build artifact. Single source of truth consumed by `generate-sitemap.mjs` (sitemap + robots) and the `llms.txt` generator.

## Schema

```jsonc
[
  {
    "path": "/",                        // string: canonical, lang-stripped, normalized (no trailing slash except root)
    "lastmod": "2026-06-07",            // string: ISO YYYY-MM-DD
    "indexable": true,                  // boolean
    "kind": "static",                   // "static" | "article"
    "title": "InvestGuideUA - ...",     // string: resolved title (<=60)
    "description": "..."                // string: resolved description (110-160)
  },
  {
    "path": "/articles/ovdp-war-bonds",
    "lastmod": "2026-06-04",            // == article.dateModified  (fixes FR-017)
    "indexable": true,
    "kind": "article",
    "title": "ОВДП: як інвестувати у військові облігації",
    "description": "Що таке ОВДП ..."
  }
]
```

## Generation rules
- One entry per public route: each `PUBLIC_PAGES` static page (incl. new `/editorial-policy`, `/contact`) + each published UK article (`/articles/<slug>`) + the `/articles` index.
- `lastmod`: article entries use `article.dateModified`; static entries use the build date (`SEO_BUILD_DATE` or today).
- `indexable`: from `isIndexable(path)`; private/utility paths are never emitted into the manifest.
- `title`/`description`: articles use frontmatter; static pages resolve `titleKey`/`descKey` against `public/i18n/uk.json` (UK is the published language).
- Deterministic order (sorted by path) for stable diffs.

## Consumers
- **sitemap**: `<loc>` from `path`, `<lastmod>` from `lastmod`, hreflang from `EN_ENABLED`.
- **llms.txt**: only `indexable` entries; grouped by `kind`.
- **robots**: unaffected by per-entry data (uses private-prefix list), but generated in the same run.

## Validation (by audit indirectly)
- A21 cross-checks sitemap `lastmod` against article `dateModified` — passes only if the manifest carried true dates.
